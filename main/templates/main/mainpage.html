{% extends 'base.html' %} {% load static %} {% block content %}

🦁2주차 세션 내용 정리🦁

1. 가상환경 설정 및 Django 설치

  • 가상환경 진입:

    pipenv shell

  • Django 설치:

    pip install django

  • .gitignore 설정 사이트: Toptal Gitignore Generator

2. 템플릿 연동 및 static 파일 설정

  • 템플릿 구조:

    templates/main/mainpage.html

  • 뷰 함수에서 render로 연결:

    render(request, 'main/mainpage.html', context)

  • {% load static %} 를 통해 이미지, CSS 파일 연결

  • 예시:
    • 이미지:

      <img src="{% static 'images/photo.jpg' %}">

    • CSS:

      <link rel="stylesheet" href="{% static 'css/style.css' %}">

    • static 파일 위치 설정 (settings.py):

      STATICFILES_DIRS = [os.path.join(BASE_DIR, 'project', 'static')]

3. 템플릿 언어 활용

  • context 데이터 출력:

    {{ 변수명 }}

  • 반복문:

    {% for item in 리스트 %} ... {% endfor %}

  • 조건문:

    {% if 조건 %} ... {% else %} ... {% endif %}

4. 템플릿 상속 및 include 구조

  • 공통 레이아웃 정의:

    base.html

  • 개별 페이지는 다음과 같이 작성: 
    {% extends 'base.html' %}
{% block content %}
    <!-- 내용 작성 -->
{% endblock %}
  • 공통 요소(Navbar 등):

    shared/_navbar.html

    로 분리하여

    {% include 'shared/_navbar.html' %}

6. 서버 실행 및 확인

  • 서버 실행:

    python manage.py runserver

  • 접속 주소:

    http://localhost:8000
